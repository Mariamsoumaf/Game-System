## SEN208 - Data Structures

# 🎮 Smart Game System

A C++ console application that recommends games and saves favorites using Tree and Linked List data structures.

---

## 📌 About the Project

This project recommends games based on user preferences using a Tree structure.
It also saves favorite games using a Linked List.

---

## 🌳 How Tree Works

The Tree asks questions to recommend a game:

- Action → Online → Call of Duty
- Action → Offline → GTA V
- Survival → Building → Minecraft
- Survival → Horror → Resident Evil

---

## 🔗 How Linked List Works

The Linked List saves favorite games:
- Add a game with player name
- View all saved games

---

## ✨ Features

- Recommend a game using Tree
- Add favorite game to Linked List
- View all saved games
- Simple and interactive menu

---

## 🛠️ Technologies

- C++
- Tree Data Structure
- Linked List Data Structure
- Code::Blocks IDE

---

## ▶️ How to Use

1. Open project in Code::Blocks
2. Press F9 to run
3. Choose from the menu
4. Get your game recommendation!
